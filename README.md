osu!曲库导出车载音乐工具
=======
将osu!曲库导出为有ID3信息的音乐
##开源库
*[IdSharp](https://github.com/judwhite/IdSharp)   .NET ID3 Tagging Library  [![License](http://img.shields.io/:license-mit-blue.svg)](http://doge.mit-license.org)

因为“Mpeg4Tag.cs”直接下载或git获取都会出现字符错误，所以需要从github网页上复制文字粘贴。“ID3v1Tag.cs”做了小修改以支持中文字符。

*[FFmpeg](http://ffmpeg.org/)

http://ffmpeg.zeranoe.com/builds/ All builds on this page are licensed as GPL 3.0

只用了exe，放到主程序文件夹下即可。
